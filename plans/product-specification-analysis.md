### 2.2 نماذج البيانات الموجودة

```typescript
// Trip - إدارة الرحلات ✅
interface Trip {
  id, name, country, city, start_date, end_date, notes, status
}

// Supplier - الموردين ✅
interface Supplier {
  id, name, company_name, city, phone, wechat_or_whatsapp,
  product_category, rating, notes, trip_id
}

// Product - المنتجات ✅
interface Product {
  id, name, oem_number, brand, size, purchase_price,
  sale_price, quantity, notes, rating, image_url
}

// PurchaseInvoice - فواتير الشراء ✅
interface PurchaseInvoice {
  id, supplier_id, trip_id, date, currency, notes, items
}

// SalesInvoice - فواتير البيع ✅
interface SalesInvoice {
  id, customer_name, date, currency, notes, items
}

// Shipment - الشحنات ✅
interface Shipment {
  id, shipment_number, shipping_company, shipping_type,
  departure_port, arrival_port, ship_date, expected_arrival_date,
  shipping_cost, weight, cartons_count, status
}

// Expense - المصروفات ✅
interface Expense {
  id, trip_id, category, amount, currency, date, notes
}

// InventoryItem - المخزون ✅
interface InventoryItem {
  id, product_name, oem_number, brand, quantity_purchased,
  quantity_sold, quantity_available, purchase_price, sale_price
}

// Quotation - عروض الأسعار ✅
interface Quotation {
  id, supplier_id, trip_id, date, notes, items
}
```

## 3. الفجوات والتحسينات المطلوبة

### 3.1 فجوات حرجة (Critical Gaps)

| # | الميزة المفقودة | التأثير | الحل المقترح |
|---|----------------|---------|-------------|
| 1 | لا يوجد backend | فقدان البيانات | إضافة Supabase/Firebase |
| 2 | Offline Mode | لا يعمل بدون إنترنت | PWA + IndexedDB |
| 3 | خريطة الموردين | لا يوجد عرض جغرافي | Mapbox/Google Maps |
| 4 | ملاحظات صوتية | فقدان طريقة إدخال سريعة | Web Speech API |
| 5 | دليل المفاوضة | لا يوجد أداة للمفاوضة | قاعدة بيانات + Audio |

### 3.2 فجوات متوسطة (Medium Gaps)

| # | الميزة المفقودة | التأثير | الحل المقترح |
|---|----------------|---------|-------------|
| 1 | البحث عن سكن | لا يوجد | صفحة سكن جديدة |
| 2 | صور المنتجات | محدودة | رفع صور محسن |
| 3 | حفظ الموقع | غير موجود | Geolocation API |
| 4 | فتح الموردين | لا يوجد نظام | نظام إحالة/Points |
| 5 | تقييمات الترجمة | غير موجود | Text-to-Speech |

### 3.3 تحسينات صغيرة (Low Priority)

| # | الميزة | التحسين |
|---|--------|---------|

## 5. توصيات فورية

### للإصدار الأول (MVP):

1. **أضف Offline Support**
   - PWA مع Service Workers
   - LocalStorage/IndexedDB للبيانات

2. **أضف دليل المفاوضة**
   - قاعدة بيانات 50-100 عبارة شائعة
   - Text-to-Speech للصينية

   - دعم الصور
